# Spylt Website UI Clone

A frontend UI clone of the Spylt website built using modern frontend technologies.  
This project focuses on recreating the visual design, layout structure, and interactive animations of the original site.

Live Demo: https://spylt-ui.vercel.app/

---

## Tech Stack

- React – Component-based UI development
- Vite – Fast development and build tool
- Tailwind CSS – Utility-first CSS framework for styling
- GSAP – Animations and UI motion effects
- JavaScript (ES6+)

---

## Features

- Responsive layout optimized for desktop and mobile screens
- Clean and structured UI inspired by the Spylt website
- Smooth animations and transitions implemented using GSAP
- Component-based architecture for better maintainability
- Modern styling using Tailwind CSS utilities

## Getting Started (Run Locally)

Follow these steps to run the project on your local machine:

```bash
# Clone the repository
git clone https://github.com/afzytk/spylt-frontend-clone.git

# Navigate to the project directory
cd spylt-frontend-clone

# Install dependencies
npm install

# Start the development server
npm run dev
```
